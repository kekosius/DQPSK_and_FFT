◆ USART_Init()

void USART_Init ( void  )

Инициализация USART канала

Выставленные параметры:

  • Baud rate - 115200
  • Parity - None
  • Stop Bit - 1
  • Data Bits - 8
  • Выбранный канал - USART1, TX - PA9, RX - PA10
  • Включено прерывание по ошибке, при прерывании вызывается USART_Reload()

void USART_Reload ( USART_T *  usart)

Перезагрузка USART канала, вызванная обработчиком ошибки

Также отправляет информационное сообщение по USART о перезагрузке. Если не удалось отправить char по USART после пяти перезагрузок подряд, то вызывается функция принудительной остановки работы Fatal_error()
