◆ LCD_DQPSK_mode()

void LCD_DQPSK_mode ( )

Переводит LCD экран в режим отображения АРСФ кодов

Закрашивает экран в белый, выводит на экран обозначение режима работы (MSG_DQPSK_1) и поля для отображения результатов АРСФ декодирования (MSG_DQPSK_2, MSG_DQPSK_3, MSG_DQPSK_4, MSG_DQPSK_5). Также сбрасывает счётчик NoResultCounter таймера TIM4.

Отображения результата декодирования АРСФ на LCD экране

Parameters
codeдекодированный hex код
speedразрешённая на участке скорость в формате десятеричного числа[XXYY], где XX - разрешенная скорость на КРЦ, YY - разрешенная скорость на СРЦ
AvgVсредняя амплитуда принятого сигнала

Если code код равен prevCode, то выводит только AvgV (предыдущие показания code, speed остаются на экране).

Также при каждом вызове обновляет счетчик NoResultCounter

Инициализация GPIO ножек для работы с LCD экраном

Дефолтные значения пинов:

  • PC4 (LCD_BL) - High

◆ swResetLCD()

software перезагрузка LCD экрана

Функция отправляет команду 0x01 по шине SPI, что вызывает software перезагрузку LCD экрана. Также, согласно документации, после перезагрузки необходимо обеспечить задержку выполнения остальных функций (установлена ~120 мс).

◆ writeCmdLCD()

void writeCmdLCD ( uint8_t  cmd)

Отправка команды на контроллер LCD экрана ILI9488.

Функция выставляет Low уровень на линии LCD_CS и линии LCD_DC, после чего выполняет отправку команды по SPI. Далее для соблюдения протокола функция выставляет High уровень на линии LCD_CS

◆ buf

◆ NoResultCounter

uint8_t NoResultCounter = 3

Зависит от количества безрезультатных (сигнал не удалось декодировать) приёмов сигнала подряд. extern переменная, уменьшается в TMR4_EventCallback() с периодом 500ms. Если NoResultCounter = 0 (т.е. в течении ~1 секунды не произошло удачного декодирования), то вызывается LCD_Show_No_Result()

◆ NoResultState

uint8_t NoResultState = 0

Индикация отсутсвия отображения кода на LCD экране. 0 - код на экране примутсвует, 1 - кода нет

◆ prevCode

uint16_t prevCode = 0x999

Содержит значение предыдущего принятого кода. 0x999 - невозможное значение, поэтому является default
