Смена режимов работы по нажатию на KEY1. 

При нажатии на кнопку KEY1 сменяет текущий режим работы. Отключает управляющий таймер для текущего режима работы, выполняет возврат к дефолтным значенияем всех используемых переменных и запускает новый режим работы. Переменная Work\+Status определяет текущий режим работы. Значение 1 соответсвует режиму АРСФ, значение 0 соответсвует режиму АРС
